Class: LocalStorageWrapper

LocalStorageWrapper

new LocalStorageWrapper(namespace)

Helper object for interfacing with the LocalStorage. The LocalStorage browser API allows for offline storage. That API is very unsophisticated, and is essentially a lame key-value store. This object sits on top and provides a number of useful utilities, including rudimentarycache cache expiration.
